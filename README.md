## Run Locally

Clone the project

```bash
  git clone https://github.com/IndraniSom/ConferenceWebsite
```

Go to the project directory

```bash
  cd ConferenceWebsite
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```
